Aqui estão alguns dos meus principais projetos em que tenho orgulho de compartilhar

  • Sistema de armazenamento de senha em python

    O presente projeto foi desenvolvido com o propósito de apresentação do Trabalho de Conclusão de Curso (TCC) relacionado à segurança no ambiente digital. Com base em pesquisas realizadas na instituição educacional em que estudei, constatou-se que muitas pessoas enfrentavam dificuldades em criar e gerenciar senhas distintas, resultando frequentemente em esquecimentos. Diante dessa problemática, decidi conceber um aplicativo que visa solucionar essa questão.

  • Sistema de tela de login com Java

    Este sistema foi desenvolvido com o propósito de treinar a utilização de SQL em conjunto com a linguagem Java. O sistema inclui uma verificação de proteção da senha no momento do cadastro, impondo os seguintes critérios: mais de 8 dígitos, pelo menos 1 letra maiúscula, pelo menos 1 letra minúscula e caracteres especiais. Durante o processo de login, o sistema verifica a existência do usuário e, caso seja confirmada, procede à verificação da senha. Se a senha fornecida coincidir com a cadastrada, o programa retorna uma mensagem de sucesso; caso contrário, uma mensagem de erro é exibida.
